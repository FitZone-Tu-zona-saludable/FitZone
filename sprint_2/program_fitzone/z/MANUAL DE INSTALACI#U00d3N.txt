MANUAL DE INSTALACIÓN - FITZONE

1. REQUISITOS
- Python 3.x
- pip

2. INSTALACIÓN

Crear entorno virtual:
python -m venv venv

Activar:
venv\Scripts\activate

Instalar dependencias:
pip install -r requirements.txt

3. EJECUCIÓN

python app.py

4. PRUEBAS

pytest

5. SONARQUBE

Instalar SonarQube y ejecutar:
sonar-scanner
